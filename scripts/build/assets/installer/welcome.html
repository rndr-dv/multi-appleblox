MultaBlox

Private AppleBlox fork for multi-account Roblox management on macOS

MultaBlox retains AppleBlox launcher features and adds private multi-instance experiments.

This installer will place MultaBlox in your Applications folder.

To uninstall, use Reset Data in Settings, then drag MultaBlox.app to the Trash.
